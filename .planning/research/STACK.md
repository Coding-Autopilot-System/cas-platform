# Stack Research

- Azure Bicep provides declarative, subscription-scope orchestration and
  resource-group-scoped modules.
- Azure Container Apps is appropriate for containerized CAS control-plane
  services with system-assigned managed identity.
- Azure Functions Flex Consumption Linux is reserved for future event-driven
  functions rather than introduced without a workload.
- Log Analytics and workspace-based Application Insights form the telemetry
  baseline.
- PowerShell and Azure CLI provide Windows-first build and safe what-if tooling.
